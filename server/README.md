

# LifeTube
Modern video sharing platform where users can upload, watch, and interact with videos.
## About
LifeTube is a full-stack video sharing application built with React and Node.js. It provides a complete video platform experience with user authentication, video management, social interactions, and a subscription system.
## Features
### Video Platform
- Upload and share videos
- Automatic thumbnail generation from videos
- Video duration tracking
- Watch videos with smooth playback
- Delete your own videos
### Social Features
- Like and dislike videos
- Comment on videos
- Reply to comments
- Subscribe to channels
- View subscription feed
- Real-time subscriber counts
### User Experience
- Clean, modern interface
- Responsive design for all devices
- Search and discover videos
- Trending videos section
- Category-based browsing
- User profiles and channels
## Tech Stack
**Frontend:** React, Vite, React Router, Axios
**Backend:** Node.js, Express, Supabase
**Database:** PostgreSQL (Supabase)
**Media:** FFmpeg for video processing
**Authentication:** JWT tokens
## Project Structure
```
├── client/          # React frontend application
└── server/          # Node.js backend API
```
